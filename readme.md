## Projeto
esse é um projeto Web Responsico de uma Cápsula do tempo para exibir memórias em uma linha do tempo. 

## Tecnologias
Esse projeto foi desenvolvido durante o NLW da Rocketsear com as seguintes tecnologias

- HTML
- CSS
- Git e Github
